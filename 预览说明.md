# 如何预览博客主页

## 方法一：使用 VS Code Live Server（推荐）

1. 在 VS Code 中安装 "Live Server" 插件
2. 右键点击 `index.html` 文件
3. 选择 "Open with Live Server"
4. 浏览器会自动打开并显示页面

## 方法二：使用 Python 启动本地服务器

在终端中运行：

```bash
cd /Users/huzx/Work/Code/abc
python3 -m http.server 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 方法三：使用 Node.js (需要安装 Node.js)

```bash
cd /Users/huzx/Work/Code/abc
npx serve .
```

## 方法四：直接拖拽文件

将 `index.html` 文件直接拖拽到浏览器窗口中（某些功能可能受限）
