## 🔧 自定义内容

### 修改文章内容

在 `ai-computing.html` 中找到 `<article class="article-item">` 标签，修改：

```html
<article class="article-item">
    <div class="article-main">
        <div class="article-meta">
            <span class="article-category">你的分类</span>
            <span class="article-date">2025-XX-XX</span>
            <span class="article-reading">XX分钟阅读</span>
        </div>
        <h3 class="article-title">
            <a href="#">你的文章标题</a>
        </h3>
        <p class="article-excerpt">
            你的文章摘要...
        </p>
        <div class="article-tags">
            <span class="mini-tag">标签1</span>
            <span class="mini-tag">标签2</span>
        </div>
        <div class="article-stats">
            <span>👀 阅读量</span>
            <span>💬 评论数</span>
            <span>👍 点赞数</span>
        </div>
    </div>
</article>
```

### 添加新文章

复制一个 `<article>` 块，修改其中的内容即可。

### 修改主题标签

在 `<div class="topic-tags">` 中添加或修改标签：

```html
<span class="topic-tag">你的标签</span>
```
