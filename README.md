# 博客主页设计

这是一个现代化、响应式的博客主页设计。

## 特性

✨ **现代设计**
- 简洁美观的UI设计
- 渐变色彩搭配
- 平滑的动画效果

📱 **完全响应式**
- 支持桌面端、平板和移动端
- 自适应布局
- 移动端优化的导航菜单

🎨 **丰富的功能**
- 精选文章展示
- 文章分类和标签
- 热门文章列表
- 搜索功能
- 邮件订阅
- 社交媒体链接
- 分页导航

⚡ **性能优化**
- 轻量级代码
- 快速加载
- 优雅的滚动效果

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可查看效果
2. 所有样式和交互功能已经完整实现
3. 图片使用了 Unsplash 的示例图片，你可以替换成自己的图片

## 自定义

### 修改颜色主题

在 `styles.css` 中修改 CSS 变量：

```css
:root {
    --primary-color: #3b82f6;      /* 主色调 */
    --secondary-color: #8b5cf6;    /* 次要色 */
    --accent-color: #06b6d4;       /* 强调色 */
}
```

### 修改内容

直接编辑 `index.html` 中的内容即可：
- 修改导航栏链接
- 更新文章内容
- 添加或删除文章卡片
- 修改个人信息

### 添加功能

在 `script.js` 中可以添加更多交互功能：
- 实际的搜索功能
- 评论系统
- 文章点赞
- 分享功能
